Introduction

Selamat datang di platform pengembang Foodify. API kami dirancang untuk memberikan akses cepat dan terstruktur data makanan Nusantara.

Gunakan API ini untuk membangun marketplace makanan nusantara. Kami menggunakan standar RESTful dengan respon berformat JSON yang dioptimalkan untuk performa tinggi.

Authentication

Setiap request ke Foodify API wajib menyertakan API Key yang valid. Anda dapat mendapatkan kunci ini melalui dashboard pengembang setelah melakukan registrasi.

⚠️ Security Note

Jangan pernah membagikan API Key Anda di sisi klien (frontend) publik. Gunakan server-side proxy jika memungkinkan untuk menjaga kerahasiaan kunci Anda.

Query Parameter

Tambahkan parameter key pada setiap URL endpoint Anda.

https://api.foodify.com/v1/resource?key=YOUR_API_KEY

Endpoints

GET Get All Foods

Endpoint ini mengembalikan daftar lengkap koleksi makanan dalam database Foodify.

Request URL
GET /api/foods?key={api_key}
Parameters
Parameter Type Status Description
key String Required API Key unik untuk mengidentifikasi aplikasi Anda.
Response Example
[ { "id": 102, "nama": "Sate Ayam Madura", "asal": "Jawa Timur", "deskripsi": "Sate Ayam Khas Madura", "gambar_url": "https://cdn.foodify.com/sate.jpg" } ]
